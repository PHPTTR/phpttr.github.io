PHPForm

การรับค่าข้อมูลและส่งผ่านฟอร์มไปประมวลผลไฟล์ที่กำหนด

1. form

ในส่วนของแท็ก form จะมี 2 คำสั่งที่เกี่ยวข้องกับการทำงาน ทำให้ form สามารถส่งข้อมูลไปหาไฟล์ที่ต้องการได้ ซึ่งก็คือคำสั่ง action และ method

action ไฟล์ที่รับข้อมูลจากฟอร์มเพื่อไปประมวลผล action="data.php"

method คือ การระบุรูปแบบการสื่อสารข้อมูล มี 2 ประเภทหลัก คือ POST และ GET

  • POST เหมาะกับการส่งข้อมูลผ่าน form, ข้อมูลที่ต้องการความปลอดภัย, ข้อมูลรูปภาพ และข้อมูลขนาดใหญ่
  • GET เหมาะกับการส่งข้อมูลไม่เยอะ ไม่เน้นความปลอดภัย เพราะการส่งแบบ GET ข้อมูลจะเป็นส่วนหนึ่งของ URL ของเว็บไซต์เรียกว่า Query string

    • 2. input

    เมื่อมีคำสั่งในการส่งข้อมูลผ่านฟอร์มและวิธีการส่งแล้วจะต้องมีเครื่องมือในการรับข้อมูล ซึ่งก็คือคำสั่ง <input type="text" name="example">

  • input มีหลากหลายประเภทในการรับข้อมูลเข้ามา type=" " text checkbox radio file password
  • name="example" ระบุชื่อของค่าที่รับเข้ามาผ่านฟอร์มมีชื่อว่าอะไร เมื่อส่งข้อมูลที่รับเข้ามาไปประมวลผล ไฟล์ที่ประมวลผลจึงรับค่าที่ส่งมาโดยอ้างอิงจากชื่อของค่าที่ส่ง

    • ประเภทของ input เบื้องต้น

  • input type text

    • รูปแบบการใช้งาน

    <input type="text" name="username">

    ผลลัพธ์

    username

  • input type password

    • รูปแบบการใช้งาน

    <input type="password" name="password">

    ผลลัพธ์

    password

  • input type radio

    • รูปแบบการใช้งาน

    <input type="radio" name="sex value="male">male
    <input type="radio" name="sex" value="female">female

    ผลลัพธ์

    male female

  • input type checkbox

    • รูปแบบการใช้งาน

    <input type="checkbox" name="langHTML" value="HTML">ภาษา HTML
    <input type="checkbox" name="langCSS" value="CSS">ภาษา CSS
    <input type="checkbox" name="langJAVA" value="JAVA Script">ภาษา JAVA Script
    <input type="checkbox" name="langPHP" value="PHP">ภาษา PHP

    ผลลัพธ์

    HTML CSS JAVA Script PHP

  • input type select

    • รูปแบบการใช้งาน

    <select name="language>
    <option value="HTML">ภาษา HTML</option>
    <option value="CSS">ภาษา CSS</option>
    <option value="JAVA">ภาษา JAVA Script</option>
    <option value="PHP">ภาษา PHP</option>
    </select>

    ผลลัพธ์


  • input type submit

    • รูปแบบการใช้งาน

    <input type="submit" value="send">

    ผลลัพธ์


  • input type reset

    • รูปแบบการใช้งาน

    <input type="reset" value="clear">

    ผลลัพธ์

    การรับค่าข้อมูลที่ส่งมาจากฟอร์ม

    ไฟล์ data.php จะรับค่าข้อมูลที่ส่งผ่านฟอร์มและนำมาประมวลผล

    สร้างตัวแปรเพื่อที่จะใช้ตัวแปรรับค่าข้อมูล $ชื่อตัวแปร = $_POST['ชื่อที่ส่งมาจากฟอร์มตาม name'];

    <?php
    $Username = $_POST['username'];
    $Password = $_POST['password'];
    $Sex = $_POST['sex'];
    $C_html = $_POST['langhtml'];
    $C_css = $_POST['langcss'];
    $C_java = $_POST['langjava'];
    $C_php = $_POST['langphp'];
    $Platform = $_POST['platform'];
    ?>

    <html>
    <body>
    <table>

    <tr>
    <td><?php echo "Your Username : ". $Username ?></td>
    </tr>

    <tr>
    <td><?php echo "Your Password : ". $Username ?></td>
    </tr>

    <tr>
    <td><?php echo "Sex : ". $Sex ?></td>
    </tr>

    <tr>
    <td><?php echo "Choose Code is : "?></td>
    <td><?php echo "$C_html " ?> </td>
    <td><?php echo "$C_css " ?> </td>
    <td><?php echo "$C_java " ?> </td>
    <td><?php echo "$C_php " ?> </td>
    </tr>

    <tr>
    <td><?php echo "Choose Platform : ". $Platform ?></td>
    </tr>

    </table>

    </body>
    </html>
