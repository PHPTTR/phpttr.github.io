ตัวดำเนินการ


1. ตัวดำเนินการทางคณิตศาสตร์

ตัวดำเนินการทางคณิตศาสตร์เป็นตัวดำเนินการในกลุ่มที่เกี่ยวข้องกับการคำนวณหาผลลัพธ์ต่างๆทางคณิตศาสตร์

ตัวดำเนินการ ตัวอย่าง เครื่องหมาย หลักการใช้
+ $x = $a + $b บวก นำ a + b และเก็บค่าไว้ใน x
- $x = $a - $b ลบ นำ a - b และเก็บค่าไว้ใน x
* $x = $a * $b คูณ นำ a * b และเก็บค่าไว้ใน x
/ $x = $a / $b หาร นำ a / b และเก็บค่าไว้ใน x
% $x = $a % $b หารเอาเศษ นำ a % b และเก็บค่าไว้ใน x

ตัวดำเนินการบวก +

<?php
$a = 10 ;
$b = 2 ;
$x = $a + $b ;

print "Value x =" . $x ;
?>

ตัวดำเนินการลบ -

<?php
$a = 10 ;
$b = 2 ;
$x = $a - $b ;

print "Value x =" . $x ;
?>

ตัวดำเนินการคูณ *

<?php
$a = 10 ;
$b = 2 ;
$x = $a * $b ;

print "Value x =" . $x ;
?>

ตัวดำเนินการหาร /

<?php
$a = 10 ;
$b = 2 ;
$x = $a / $b ;

print "Value x =" . $x ;
?>

ตัวดำเนินการหารเอาเศษ %

<?php
$a = 10 ;
$b = 2 ;
$x = $a % $b ;

print "Value x =" . $x ;
?>

2. ตัวดำเนินการเพิ่มและลดค่า

ตัวดำเนินการเพิ่มและลดค่าของภาษา PHP ใช้เพื่อเพิ่มหรือลดค่าของตัวแปร

ตัวดำเนินการ ตัวอย่าง หลักการใช้
++ $a++ รีเทิร์นค่า a หรือก็คือตัวแปรมีค่าคงเดิมแล้วจึงเพิ่มค่าตัวแปรทีละหนึ่ง
++ ++$a เพิ่มค่า a ทีละหนึ่ง
-- $a-- รีเทิร์นค่า a หรือก็คือตัวแปรมีค่าคงเดิมแล้วจึงลดค่าตัวแปรทีละหนึ่ง
-- --$a ลดค่า a ทีละหนึ่ง

ตัวดำเนินการเพิ่มค่า รีเทิร์นค่าหรือตัวแปรมีค่าคงเดิมแล้วจึงเพิ่ม

<?php
$x = 10 ;
$x++ ;
print $x ;
?>

ตัวดำเนินการเพิ่มค่า

<?php
$x = 10 ;
print ++$x ;
?>

ตัวดำเนินการลดค่า รีเทิร์นค่าหรือตัวแปรมีค่าคงเดิมแล้วจึงลด

<?php
$x = 10 ;
$x-- ;
print $x ;
?>

ตัวดำเนินการลดค่า

<?php
$x = 10 ;
print --$x ;
?>

3. ตัวดำเนินการกำหนดค่า

ตัวดำเนินการกำหนดค่าจะทำการเอาตัวแปรไปดำเนินการทางคณิตศาสตร์กับค่าที่กำหนด

ตัวดำเนินการ ตัวอย่าง หลักการใช้ รูปแบบประโยคสัญลักษณ์
+= $a += 10 กำหนดค่าให้ a แล้วนำ a บวกกับ 10 $a = $a + 10
-= $a -= 10 กำหนดค่าให้ a แล้วนำ a ลบกับ 10 $a = $a - 10
*= $a *= 10 กำหนดค่าให้ a แล้วนำ a คูณกับ 10 $a = $a * 10
/= $a /= 10 กำหนดค่าให้ a แล้วนำ a หารกับ 10 $a = $a / 10
%= $a %= 10 กำหนดค่าให้ a แล้วนำ a หารเอาเศษกับ 10 $a = $a % 10

ตัวดำเนินการกำหนดค่าให้ a แล้วนำ a + 10

รูปแบบประโยคสัญลักษณ์ $a = $a + 10

<?php
$a = 100 ;
$a += 10 ;
print $a ;
?>

ตัวดำเนินการกำหนดค่าให้ a แล้วนำ a - 10

รูปแบบประโยคสัญลักษณ์ $a = $a - 10

<?php
$a = 100 ;
$a -= 10 ;
print $a ;
?>

ตัวดำเนินการกำหนดค่าให้ a แล้วนำ a * 10

รูปแบบประโยคสัญลักษณ์ $a = $a * 10

<?php
$a = 100 ;
$a *= 10 ;
print $a ;
?>

ตัวดำเนินการกำหนดค่าให้ a แล้วนำ a / 10

รูปแบบประโยคสัญลักษณ์ $a = $a / 10

<?php
$a = 100 ;
$a /= 10 ;
print $a ;
?>

ตัวดำเนินการกำหนดค่าให้ a แล้วนำ a % 10

รูปแบบประโยคสัญลักษณ์ $a = $a % 10

<?php
$a = 100 ;
$a %= 10 ;
print $a ;
?>

4. ตัวดำเนินการเปรียบเทียบ

ตัวดำเนินการเปรียบเทียบที่ใช้สำหรับเปรียบเทียบระหว่างค่าสองค่า ผลลัพธ์ของการเปรียบเทียบจะได้เป็น Boolean ที่มีค่าเป็นเพียง True หรือ False

ตัวดำเนินการ ตัวอย่าง ผลลัพธ์ หลักการใช้
== $a == $b เป็นจริงถ้า a เท่ากับ b == คือการเปรียบเทียบว่าค่าเท่ากันหรือไม่
!= , <> $a != $b เป็นจริงถ้า a ไม่เท่ากับ b != , <> คือการเปรียบเทียบว่าค่าไม่เท่ากันหรือไม่
< $a < $b เป็นจริงถ้า a น้อยกว่า b < คือการเปรียบเทียบว่าค่า a น้อยกว่า bหรือไม่
<= $a <= $b เป็นจริงถ้า a น้อยกว่าหรือเท่ากับ b <= คือการเปรียบเทียบว่าค่า a น้อยกว่าหรือเท่ากับ bหรือไม่
> $a > $b เป็นจริงถ้า a มากกว่า b > คือการเปรียบเทียบว่าค่า a มากกว่า bหรือไม่
>= $a >= $b เป็นจริงถ้า a มากกว่าหรือเท่ากับ b >= คือการเปรียบเทียบว่าค่า a มากกว่าหรือเท่ากับ bหรือไม่

เปรียบเทียบค่า a เท่ากับ b หรือไม่

เป็นจริงถ้า a เท่ากับ b เป็นเท็จถ้าไม่เท่ากัน

<?php
$a = 10 ;
$b = 10 ;

if ( $a == $b) {
print "True" ;
}
else {
print "False" ;
}
?>

เปรียบเทียบค่า a ไม่เท่ากับ b หรือไม่

เป็นจริงถ้า a ไม่เท่ากับ b เป็นเท็จถ้าเท่ากัน

<?php
$a = 20 ;
$b = 10 ;

if ( $a != $b) {
print "True" ;
}
else {
print "False" ;
}
?>

เปรียบเทียบค่า a น้อยกว่า b หรือไม่

เป็นจริงถ้า a น้อยกว่า b เป็นเท็จถ้า a มากกว่า b

<?php
$a = 9 ;
$b = 10 ;

if ( $a < $b) {
print "True" ;
}
else {
print "False" ;
}
?>

เปรียบเทียบค่า a น้อยกว่าเท่ากับ b หรือไม่

เป็นจริงถ้า a น้อยกว่าหรือเท่ากับ b เป็นเท็จถ้า a มากกว่า b

<?php
$a = 10 ;
$b = 10 ;

if ( $a <= $b) {
print "True" ;
}
else {
print "False" ;
}
?>

เปรียบเทียบค่า a มากกว่า b หรือไม่

เป็นจริงถ้า a มากกว่า b เป็นเท็จถ้า a น้อยกว่า b

<?php
$a = 20 ;
$b = 10 ;

if ( $a > $b) {
print "True" ;
}
else {
print "False" ;
}
?>

เปรียบเทียบค่า a มากกว่าเท่ากับ b หรือไม่

เป็นจริงถ้า a มากกว่าหรือเท่ากับ b เป็นเท็จถ้า a น้อยกว่า b

<?php
$a = 10 ;
$b = 10 ;

if ( $a >= $b) {
print "True" ;
}
else {
print "False" ;
}
?>

5. ตัวดำเนินการตรรกะ

ตัวดำเนินการตรรกะ คือ ตัวดำเนินการที่ใช้สำหรับเชื่อมต่อค่าความจริง

ตัวดำเนินการ ตัวอย่าง หลักการใช้
และ ( and ) ($a and $b) นำค่า a และ b มาเปรียบเทียบจะต้องเป็นจริงทั้งคู่
หรือ ( or ) ($a or $b) นำค่า a และ b มาเปรียบเทียบจะต้องเป็นจริงเพียงตัวใดตัวหนึ่ง แต่จะเป็นเท็จถ้าทั้ง a และ b เป็นเท็จทั้งคู่
นิเสธ ( ! ) ($a !== $b) นำค่า a และ b มาเปรียบเทียบจะมีค่าเป็นจริงก็ต่อเมื่อเปรียบเทียบแล้วเป็นเท็จ

เปรียบเทียบค่า a และ b เป็นจริงหรือไม่ ค่า a และ b จะต้องเป็นจริงทั้งคู่ในตรรกะ และ ถึงจะเป็นค่าจริง

กำหนดค่าให้ a = 10<20 ซึ่งก็คือค่า จริง

กำหนดค่าให้ b = 20<10 ซึ่งก็คือค่า จริง

<?php
$a = 10<20 ;
$b = 20>10 ;

if ( $a and $b ) {
 print "True" ;
}
else {
print "False" ;
}
?>

เปรียบเทียบค่า a หรือ b เป็นจริงหรือไม่ ค่า a หรือ b จะต้องเป็นจริงเพียงตัวใดตัวหนึ่งในตรรกะ หรือ ถึงจะเป็นค่าจริง

กำหนดค่าให้ a = 10<20 ซึ่งก็คือค่า จริง

กำหนดค่าให้ b = 20<10 ซึ่งก็คือค่า เท็จ

<?php
$a = 10<20 ;
$b = 20<10 ;

if ( $a or $b ) {
 print "True" ;
}
else {
print "False" ;
}
?>

เปรียบเทียบค่า a ไม่เท่ากับ b หรือไม่ การเปรียบเทียบจะต้องต้องเป็นเท็จจึงจะรีเทิร์นค่าเป็นจริง

กำหนดค่า a = 10 b = 20 เมื่อเปรียบเทียบ ($a !== $b ) จะได้ค่าเป็นจริง

<?php
$a = 10 ;
$b = 20 ;

if ( $a !== $b ) {
 print "True" ;
}
else {
print "False" ;
}
?>
