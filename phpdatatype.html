ตัวแปรและชนิดของข้อมูล


1.กฏการตั้งชื่อตัวแปร

  • การประกาศตัวแปรจะต้องขึ้นต้นด้วย $
  • ไม่ขึ้นต้นด้วยตัวเลข แต่สามารถใช้ตัวเลขประกอบในชื่อของตัวแปรได้
  • สามารถใช้ตัวอักษรภาษาอังกฤษ หรือภาษาไทยได้
  • สามารถใช้สัญลักษณ์ _ (underscore) ขึ้นต้น หรือประกอบในชื่อของตัวแปรได้
  • ไม่ใช้สัญลักษณ์ทางคณิตศาสตร์ประกอบในชื่อของตัวแปร
  • ชื่อตัวแปรห้ามมีเว้นวรรค
  • การประกาศตัวแปรนั้น การใช้ตัวอักษรพิมพ์เล็กหรือพิมพ์ใหญ่ แม้จะสะกดเหมือนกันแต่ถือว่าเป็นคนละตัวแปรกัน ซึ่ง PHP มีลักษณะเป็น Case sensitive

    • 2.ชนิดข้อมูล

    ข้อมูลชนิดกลุ่มอักขระ (string)

    ข้อมูลที่นำเอาตัวอักษร ตัวเลข และเครื่องหมายต่าง ๆ มาประกอบกันเป็นข้อความ

    <?php
    $day = "Friday" ;
    $time = "08:00" ;

    print "Date" . $day . "<br>" ;
    print "Time" . $time ;
    ?>

    ข้อมูลชนิดตรรกะ (boolean)

    ข้อมูลที่เก็บค่า true (จริง) หรือ false (เท็จ) ตัวแปรชนิดนี้ส่วนใหญ่เอาไปใช้ในการเปรียบเทียบในเงื่อนไขเพือตัดสินใจดำเนินการขั้นต่อไป

    <?php
    $x = false ;

    if ($x) {
    print "success" ;
    }
    else {
    print "fail" ;
    }
    ?>

    ข้อมูลชนิดเลขจำนวนเต็ม (Integer)

    ข้อมูลชนิดเลขจำนวนเต็ม ทั้งจำนวนเต็มบวก จำนวนเต็มลบ และจำนวนศูนย์

    <?php
    $x = 10 ;
    $y= 5 ;
    $z ;
    $z = $x + $y ;

    print "Value $z = " . $z ;
    ?>

    ข้อมูลชนิดเลขทศนิยม (Floating-Point Number)

    ข้อมูลชนิดเลขทศนิยม โดยอาจเป็นค่าทศนิยมที่เป็นจำนวนบวก จำนวนลบ หรือรูปแบบตัวเลขทางวิทยาศาสตร์

    <?php
    $x = 1.5 ;
    $y= 1500 ;

    print $x ."kilometer =" . $y . "meter" ;
    ?>

    ข้อมูลชนิดอาร์เรย์ (Array)

    อาร์เรย์คือการเก็บข้อมูลชนิดเดียวกัน โดยสามารถเก็บค่าข้อมูลได้หลายค่า โดยข้อมูลใน () จะเรียกว่า index เริ่มตั้งแต่ 0 เช่น $num[0] , $num[1] , $num[2] , $num[3]

    <?php
    $num = array(1.5,7,-9,"Hello",true) ;

    print $num[0] . "<br>" ;
    print $num[1] . "<br>" ;
    print $num[2] . "<br>" ;
    print $num[3] . "<br>" ;
    print $num[4] . "<br>" ;
    var_dump($num);
    ?>
