phpMyAdmin คืออะไร

โปรแกรมที่ถูกพัฒนาโดยใช้ภาษา PHP เพื่อใช้ในการบริหารจัดการฐานข้อมูล Mysql ผ่านเว็บเบราว์เซอร์

เริ่มต้นการใช้งาน phpMyAdmin

1. download โปรแกรมที่จำลองเครื่องคอมพิวเตอร์ให้เป็น web server เช่น Appserv , XAMPP

2. พิมพ์ http://localhost/phpmyadmin/ ที่ URL บน web browser

3. คลิกที่ phpMyAdmin เพื่อทำการกรอกข้อมูล username และ password

username = root

password = 12345678


เริ่มต้นการสร้าง Database

ทำการสร้าง Database โดยคลิกตามลำดับ

1. คลิกเลือกที่ new เพื่อสร้าง Database ใหม่

2. ตั้งชื่อ Database

3. คลิก Create เพื่อสร้าง Database


เริ่มต้นการสร้างตาราง (Table)

ทำการสร้างตาราง (Table) โดยคลิกตามลำดับ

1. ตั้งชื่อตาราง Table name

2. ตั้งชื่อ Column

3. กำหนดค่าของ Column นั้นว่าค่าที่เก็บเป็นตัวอักษรหรือตัวเลข

4. กำหนดขนาดของ Column จะเก็บขนาดเท่าไหร่

5. กำหนด Column แรกเป็น Primary Key

6. คลิก Save เพื่อทำการสร้างตาราง (Table)


เริ่มต้นการแทรกข้อมูลลงในตาราง (Table)

เมื่อทำการสร้างตาราง (Table) ให้แทรกข้อมูลลงในตาราง โดยคลิกตามลำดับ

1. เลือก Insert

2. Value ตือค่าข้อมูลที่ต้องการแทรกในแต่ละคอลัมน์

3 เมื่อกรอกข้อมูลเรียบร้อย ให้คลิก Go เพื่อทำการแทรกข้อมูลลงในตาราง


สร้างไฟล์ connect.php สำหรับเชื่อมต่อฐานข้อมูล MySQL

สร้างตัวแปร $con_nect เพื่อเก็บฟังก์ชันในการเชื่อมต่อฐานข้อมูล = mysqli_connect("hostname","username","password","database name");

ในการเช็คฟังก์ชันการเชื่อมต่อฐานข้อมูลว่าเชื่อมต่อได้หรือไม่ สามารถใช้ตัวแปรที่สร้างเก็บค่าฟังก์ชัน $con_nect และคำสั่ง if($con_nect -> connect_eror)

die คือคำสั่งถ้าไม่สามารถเชื่อมต่อฐานข้อมูลได้จะทำงานในบล็อกการทำงานในส่วน die

<?php
$con_nect = mysqli_connect("localhost","root","12345678","product");
if($con_nect -> connect_error){
die("Connection failed");
}
echo "Connected successfully";
?>
