คำสั่งในการทำซ้ำหรือการวนลูป


1. for

for (1. กำหนดตัวแปรสำหรับเริ่มต้น $x; 2. สร้างเงื่อนไขถ้าตรงเงื่อนไขจะยังวนลูป $x<=10; 3. เพิ่มหรือลดค่าตัวแปรไปเรื่อยๆจนกว่าจะพ้นเงื่อนไข $x++)
{ คำสั่ง }

การทำงานจะเรียงลำดับดังนี้

  • ขั้นแรกกำหนดค่า $x = 0;
  • นำตัวแปรไปเปรียบเทียบในเงื่อนไขตามคำสั่ง $x<=10; ถ้าเป็นจริงให้ทำตามคำสั่งในบล็อก { } แต่ถ้าเป็นเท็จจึงจะออกจากการทำงานวนลูป

  • เมื่อตรวจสอบเงื่อนไขว่าเป็นจริงแล้วทำงานตามคำสั่งในบล็อก { }

  • เมื่อทำงานตามคำสั่งในบล็อก { } จึงจะเพิ่มค่าตามคำสั่ง $x++ เมื่อเพิ่มค่าเสร็จจะวนลูปไปที่เงื่อนไขเพื่อเปรียบเทียบว่าเป็นจริงหรือเท็จ
    • <?php
    for ($x =0; $x>= 10; $x++) {
    print "number is : $x <br>" ;
    }
    ?>

    2. while

    while (เงื่อนไข)
    { คำสั่ง }

    การทำงานจะเรียงลำดับดังนี้

  • กำหนดตัวแปร $x=10;
  • เมื่อวนลูปจะนำตัวแปรไปเปรียบเทียบเงื่อนไขตามคำสั่ง $x>=0 ถ้าเป็นจริงให้ทำตามคำสั่งในบล็อก { }

  • เมื่อทำตามคำสั่งในบล็อก { } แล้วจึงจดลงค่าตัวแปรทีละหนึ่งตามคำสั่ง $x--;
    • <?php
    $x= 10;
    while ($x>= 0 ) {
    print "number is : $x <br>" ;
    $x--;
    }
    ?>

    3. do while

    คำสั่ง do while จะทำงานในลักษณะ ทำก่อนแล้วเช็คเงื่อนไขทีหลัง

    do {
    คำสั่ง
    $x++;
    }

    while (เงื่อนไข)

    การทำงานจะเรียงลำดับดังนี้

  • กำหนดตัวแปร $x=1;
  • คำสั่ง do คือสั่งให้ทำก่อนหนึ่งครั้ง คือ print ค่าออกมา แล้วจึงเพิ่มค่าตัวแปรทีละหนึ่งตามคำสั่ง $x++;

  • เมื่อเพิ่มค่าตัวแปรเสร็จจะไปเช็คเงื่อนไขในลูป while ว่าเป็นจริงหรือเท็จ ถ้าเป็นจริงจะย้อนกลับไปทำในลูป do
    • <?php
    $x= 1;
    do {
    print "number is : $x <br>" ;
    $x++;
    }
    while($x<=10)
    ?>
